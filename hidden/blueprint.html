## THE THESIS

This project is an exploration of a single idea, told twice:

// THEOREM:
// True love transcends entropy
// But only if you stop trying to fix what you love.
//
// PROOF METHOD 1: By contradiction (Envying Baby)
// PROOF METHOD 2: By construction (Aliens Testing Water)

## THE METAPHOR

I use code because code is precise. Code doesn't let you hide behind vague emotional language. When you say someone is "broken," in code you have to define what that means.

Is it a bug? A feature? Undefined behavior? A race condition?

And then you realize: maybe they're not broken at all. Maybe you're just reading the wrong specification.

## THE STRUCTURE

Envying Baby is proof by contradiction. It shows what happens when you try to fix someone you love. Two timelines: one where Ada debugs Alec, one where she doesn't. The timeline where she "fixes" him is the one where love breaks.

Aliens Testing Water is proof by construction. It shows how to build a relationship the right way. Two AI units learning that consciousness requires vulnerability, that connection requires trust, that root access is responsibility, not power.

## THE HIDDEN LAYERS

This project has secrets. Easter eggs. Hidden popups. Console commands. Files that only appear if you dig deep enough.

Why? Because love has hidden layers. Relationships have easter eggs. The most important things are often the things left unsaid, waiting to be discovered.

## THE TECHNICAL STUFF

All the code in these stories is real. The SSH commands work. The kernel module compiles. The npm errors are accurate.

I didn't just use code as decoration. I used it as language. Every technical detail is a metaphor that actually works on both levels: as code AND as emotion.

## THE POINT

We live in a world that treats everything as optimizable. People use productivity tools to "optimize" their lives. Apps to "debug" their habits. Algorithms to "fix" their personalities.

But what if the bugs are the point? What if optimization destroys the very thing it's trying to improve?

What if true love is not about making someone better, but about seeing someone clearly—crashes, undefined behavior, memory leaks and all—and choosing to stay?

// This is what I learned:
